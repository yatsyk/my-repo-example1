# Getting Started

## Installation

```bash
# Install my-repo-example
npm install
```

## Usage

Add your documentation as Markdown files in the `docs/` directory.
Each `.md` file becomes a page on your site.

## Directory Structure

```
docs/
  index.md          # Home page
  getting-started.md # This page
  guides/
    my-guide.md     # Nested pages become sections
```
